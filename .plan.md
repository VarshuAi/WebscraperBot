# Implementation Plan for WebscraperBot

## Architecture

### Files
- `webscraper-bot/bot.py` — Main bot (telebot + menu system + session state)
- `webscraper-bot/scraper.py` — All scraping logic (BS4 + Playwright)
- `webscraper-bot/crawler.py` — BFS web crawler with depth control
- `webscraper-bot/requirements.txt`
- `webscraper-bot/.env.example`
- `webscraper-bot/README.md`

### Stack
- `pyTelegramBotAPI` — bot framework (polling)
- `requests` + `beautifulsoup4` + `lxml` — fast static scraping
- `playwright` — screenshot, recording, cookies, localStorage (JS rendering)
- `asyncio` + thread executor — Playwright in sync bot

### Scraping Modes
1. Full Content — visible text extraction
2. HTML Data — raw HTML source
3. All Links — href extraction (absolute + relative → absolute)
4. All Paragraphs — <p> tags text
5. All Images — img src (absolute)
6. All Audio — audio/source tags + regex mp3/ogg/wav
7. All Video — video/iframe tags + regex mp4/webm/m3u8
8. All PDFs — <a href="*.pdf"> + regex
9. Cookies — Playwright context.cookies()
10. LocalStorage — Playwright page.evaluate(localStorage)
11. Metadata — title, meta tags, OG tags, headers
12. Screenshot — Playwright full-page PNG
13. Recording — Playwright video recording (webm → sent as file)
14. Web Crawl — BFS crawler, same-domain, depth=3, max 50 pages
