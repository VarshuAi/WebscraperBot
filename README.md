<!-- ============================== COMMANDS ============================== -->

<h2>
<img src="https://media.giphy.com/media/iY8CRBdQXODJSCERIr/giphy.gif" width="28">
<samp>&nbsp;COMMANDS</samp>
</h2>

```bash
# ── Basic ─────────────────────────────────────────────
/start               →  Show welcome menu
/help                →  Show help
/visits              →  Show your request count

# ── Scraping ──────────────────────────────────────────
https://example.com  →  Set URL + open scrape menu

/url <url>           →  Set URL via command

# ── Web Crawl ─────────────────────────────────────────
/crawl <url>                       →  Crawl with defaults (depth=3, pages=50)
/crawl <url> <depth> <max_pages>   →  Custom crawl settings
```

<!-- ============================== HOW IT WORKS ============================== -->

<h2>
<img src="https://media.giphy.com/media/j2pOGeGYKe2xCCKwfi/giphy.gif" width="30">
<samp>&nbsp;HOW IT WORKS</samp>
</h2>

```
STEP 1  →  Send a URL to the bot
             https://target.com

STEP 2  →  Bot shows inline menu with 14 options

STEP 3  →  Tap any option
             📄 Full Content  /  🔗 Links  /  📸 Screenshot  /  🕸️ Crawl ...

STEP 4  →  Results delivered instantly
             • Small results → sent as formatted message
             • Large results → auto-sent as .txt / .json / .png / .webm file

STEP 5  →  Send another URL anytime to change target
```

<!-- ============================== SETUP ============================== -->

<h2>
<img src="https://media.giphy.com/media/LnQjpWaON8nhr21vNW/giphy.gif" width="28">
<samp>&nbsp;SETUP</samp>
</h2>

```bash
# 1. Clone
git clone https://github.com/VarshuAi/WebscraperBot.git
cd WebscraperBot

# 2. Install dependencies
pip install -r requirements.txt

# 3. Install Playwright browser
playwright install chromium

# 4. Configure
cp .env.example .env
# → Edit .env → set BOT_TOKEN=your_token_here

# 5. Run 🚀
python bot.py
```

<!-- ============================== CRAWLER DETAILS ============================== -->

<h2>
<samp>&nbsp;🕸️ WEB CRAWL DETAILS</samp>
</h2>

```yaml
algorithm: BFS (Breadth-First Search)
scope: Same-domain only (no external links)
defaults:
  max_pages: 50
  max_depth: 3

output:
  - Live progress updates (every 5 pages)
  - Summary with full stats
  - crawl_pages.txt  → all crawled page titles + URLs
  - crawl_report.json → complete report (pages, links, images)

custom_crawl:
  command: "/crawl https://example.com 2 20"
  # depth=2, max_pages=20
```
